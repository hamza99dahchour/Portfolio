# Portfolio — Hamza Dahchour

Site personnel one-page en français, construit sur le même modèle que le portfolio de Mourad Harzallah (mêmes sections : À propos, Expérience, Projets, Compétences, Activités, Articles, Témoignage, Contact), avec une charte graphique claire (crème / vert foncé / orange) reprenant le design de vos maquettes.

## Fichiers

- `index.html` — page principale
- `article1.html`, `article2.html`, `article3.html` — articles de blog liés à votre mémoire et à votre parcours
- `hero-portrait.png` — illustration de la photo de couverture (extraite de votre maquette)
- `CV_Hamza_Dahchour_Data_Scientist.pdf` — votre CV, téléchargeable depuis le site

## À personnaliser avant mise en ligne

1. **LinkedIn** : remplacez `https://linkedin.com/in/hamza-dahchour` par votre vraie URL LinkedIn (présent dans le hero et la section contact).
2. **Formulaire de contact** : le formulaire pointe vers `https://formspree.io/f/VOTRE_ID_FORMSPREE`. Créez un compte gratuit sur [formspree.io](https://formspree.io), créez un formulaire, et remplacez l'URL par la vôtre.
3. **Témoignage** : le témoignage affiché est un exemple à remplacer par une vraie citation d'un maître de stage ou tuteur (avec son accord).
4. **Photo** : `hero-portrait.png` est une image recadrée depuis votre capture d'écran. Si vous avez le fichier original de l'illustration (ou souhaitez utiliser une vraie photo), remplacez ce fichier en gardant le même nom.
5. **Liens GitHub des projets** : les cartes "Détection de fraude" et "Système décisionnel achats & commandes" pointent vers `#`. Remplacez par les liens vers vos dépôts GitHub réels une fois publiés.
6. **Dates de l'alternance Carrefour** : ajustez si besoin les dates exactes (actuellement "2025 – 2026").

## Mise en ligne sur GitHub Pages

1. Créez un dépôt nommé exactement `VOTRE-PSEUDO-GITHUB.github.io` (remplacez par votre pseudo GitHub).
2. Ajoutez tous les fichiers de ce dossier à la racine du dépôt.
3. Poussez sur la branche `main`.
4. Dans **Settings → Pages**, vérifiez que la source est bien la branche `main` / dossier racine.
5. Votre site sera disponible à `https://VOTRE-PSEUDO-GITHUB.github.io` en quelques minutes.
